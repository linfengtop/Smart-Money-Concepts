# 复盘指南

## 单笔复盘

按这个顺序写：

1. 客观环境：高周期结构、区间、目标流动性。
2. 触发事实：扫了哪里、何时出现位移、突破哪个 Swing。
3. 执行数据：入场、止损、目标、仓位、实际盈亏和 R。
4. 行为评价：是否按计划。
5. 唯一改进点：下次可执行的具体动作。

避免使用“感觉不对”“主力骗线”“运气不好”作为结论。

## 错误标签

- 无计划入场
- 追价
- 逆高周期
- 区间中部
- 未扫流动性
- 无位移
- 无 MSS
- FVG 反复穿越
- 止损过宽
- 移动止损扩大风险
- 浮亏加仓
- 对冲代替止损
- 情绪提前平仓
- 过度交易
- 事件前冒险

一笔交易可记录多个标签，但周复盘只重点处理频率最高或损失最大的一个错误。

## 每周复盘

统计：

- 总交易数和合格模型数；
- 胜率；
- 平均盈利 R、平均亏损 R；
- 期望值；
- Profit Factor；
- 最大回撤；
- 最大连续亏损；
- 按计划执行率；
- 不同时段、方向和模型的表现。

期望值：

```text
Expectancy(R) = 胜率 × 平均盈利R - 败率 × 平均亏损R绝对值
```

## 诊断顺序

1. 先检查是否按规则执行。
2. 再检查样本量是否足够。
3. 然后检查市场环境分类。
4. 最后才修改入场或退出参数。

执行率低时，不应根据混杂数据否定策略，也不应继续增加新概念。
