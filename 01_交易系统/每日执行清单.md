# 每日执行清单

## 盘前

- [ ] 标记 Previous Day High / Low。
- [ ] 标记 Previous Week High / Low。
- [ ] 标记亚洲时段高低点和最近等高/等低。
- [ ] 画出当前 4H 或 1H Dealing Range。
- [ ] 判断 Premium、Equilibrium、Discount。
- [ ] 写出高周期偏向：多头 / 空头 / 中性。
- [ ] 写出价格可能获取的上方和下方流动性。
- [ ] 查看重大宏观事件时间。
- [ ] 查看 Funding、OI、现货与永续价差，只作为环境信息。
- [ ] 写出“今天不交易”的条件。

## 入场前

- [ ] 价格位于有意义的位置，而不是区间中部。
- [ ] 已经扫过明确流动性。
- [ ] 扫流动性后发生快速收回。
- [ ] 出现明显 Displacement。
- [ ] Displacement 形成有效 MSS。
- [ ] FVG 来自这次位移。
- [ ] 是第一次合理回踩，而不是反复穿越的旧区域。
- [ ] 止损对应客观失效点。
- [ ] 目标对应明确流动性。
- [ ] 计划盈亏比达到最低标准。
- [ ] 仓位按账户风险计算完成。
- [ ] 没有达到当日亏损或连续亏损上限。

## 持仓中

- [ ] 不移动止损扩大风险。
- [ ] 不用反向对冲代替止损。
- [ ] 不在浮亏中临时摊平。
- [ ] 只按照预先写明的条件减仓或保护。

## 平仓后

- [ ] 填写实际盈亏和 R 倍数。
- [ ] 记录是否按计划执行。
- [ ] 选择违规标签。
- [ ] 保存入场前、入场后和结束后三张截图。
- [ ] 用一句话说明这笔交易最值得保留或修正的行为。
